#  Starknet Canvas

## Development

Run starknet node

```bash
./env/bin/activate   # activate python virtual env
nile node
```

Install frontend dependencies

```bash
yarn
```

Run frontend locally

```bash
yarn dev
```

#### Run tests

```bash
yarn test
```
